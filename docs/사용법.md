# dawn-cut 사용법

> 현재 빌드 기준 실측 작성. 100% 로컬(영상이 이 Mac을 떠나지 않음).

## 0. 두 가지 입력 — 무엇을 넣느냐로 쓸 수 있는 게 갈린다

| 넣는 영상 | 풀 파워 |
|---|---|
| **말소리 있는 영상**(강의·해설·인터뷰) | 자동 자막 + 말버릇/무음 제거 + 검수 + 자동 하이라이트 + 색보정/리프레이밍/오버레이 + 내보내기 — **전부** |
| **소리 없는 화면녹화**(앱 시연·슬라이드) | 자막 계열은 불가(소리가 없으니). 색보정·9:16·오버레이·컷·내보내기는 됨 |

→ 강사님 강의 영상(0530/0601 폴더)이 **풀 파워 대상**입니다.

```
[상단 툴바]  가져오기 · 열기 · 저장 │ 되돌리기 · 다시하기 │ 무음 제거 · ⚙ │ 삭제(N) │ 내보내기 ▾
[좌측 레일]  🎬 미디어 │ 🎚 효과   (고급 모드: + 🗣 음성·TTS, ✨ 스티커·GIF)

[하단]       TIMELINE (VIDEO / OVERLAY / VOICE 트랙)
```

## 2. 기본 워크플로 (강의 자막 달기)

1. **가져오기** — 상단 `가져오기` 버튼으로 강의 mp4 선택(또는 좌측 미디어 패널에 **드래그앤드롭**). 미리보기에 영상이 뜬다. *(큰/고해상도 영상은 "변환 중 → 프록시"로 잠깐 바뀐 뒤 재생됨 — 정상.)*
2. **자막 생성** — 우측 대본 패널의 **`🎙 자막 생성 (받아쓰기)`** 클릭 → 오디오 추출 + 한국어 받아쓰기. 끝나면 어절이 대본에 나타난다. *(소리 없는 영상이면 이 버튼이 비활성 + 안내가 뜬다.)*
3. **다듬기**(선택):
   - **🧹 말버릇 N개 제거** — "어/음/그" 군더더기 일괄 컷(절약 시간 표시).
   - **🔎 검수** 토글 — STT가 헷갈린 어절을 빨갛게 표시 → **어절 더블클릭으로 교정** → "다음 의심 어절"로 이동.
   - **텍스트 컷** — 대본에서 어절 클릭(선택) → `Delete`/`Backspace` → 그 구간 영상이 잘림.
4. **자막 입히기** — 대본 패널 상단 **`자막 입히기`** → 영상에 자막이 번인된다(다시 누르면 해제).
5. **내보내기** — 상단 **`내보내기 ▾`** → **MP4**(자막 포함) / **GIF** / **SRT**(자막 파일만). 또는 **저장**으로 `.dawn` 프로젝트 파일 보관(나중에 `열기`로 복원).

## 3. 기능별 사용법

### 미디어
- **가져오기**: 툴바 `가져오기` 또는 미디어 패널 드래그앤드롭. 영상=소스, 이미지/GIF=오버레이.
- **치우기**: 미디어 카드 오른쪽 **✕** → 영상·자막·편집 전부 비움.
- 카드 배지: `원본`(그대로 재생) / `프록시`(미리보기용 변환본) / `변환 중`.

### 미리보기 / 이동
- **▶** 재생/일시정지(`Space`도 동일).
- **스크럽 바** 또는 **하단 타임라인 클릭** → 그 시점으로 이동(이동하면 일시정지 → ▶로 재생).
- `←/→` 0.1초 이동, `Home/End` 처음/끝. 대본 어절 **⌘+클릭** → 그 어절 위치로 점프.

### 효과 탭 (좌측 🎚 효과)
- **✨ 자동 보정 (1탭)** — 영상을 분석해 밝기·대비·채도 자동 보정.
- **색보정 프리셋** — 화사하게(vivid)/따뜻하게/차갑게/선명하게/시네마틱/플랫.
- **비율(Reframe)** — 원본 / **세로 9:16(쇼츠)** / 정사각 1:1 (내보낼 때 중앙 크롭).

### 무음 제거 (툴바)
- **무음 제거** 버튼으로 한 번에. 옆 **⚙** → 민감도(dB)·최소 무음 길이 조절 + 절약 미터 미리보기.

### 스타일 팩 (대본 패널 🎨)
- 드롭다운에서 1클릭 → 색+자막스타일+말버릇 정리가 한 번에(viral-punch / mukbang-sizzle / beauty-glow / golden-hour / city-night / talk-clean).

## 4. 고급 모드 기능 (`DAWN_ADVANCED=1`로 실행 시)

기본(단순) 모드에선 숨겨져 있고, **고급 모드**에서 나타난다:

- **✂️ 자동 하이라이트 (60초)** — 긴 강의를 핵심만 남긴 짧은 클립으로(롱폼→쇼츠).
- **🤖 자연어 편집(NL 바)** — `"말버릇 빼고 60초 하이라이트로 핵심 강조"`처럼 한 문장 → 제안 카드 → 승인. *(로컬 LLM `pnpm setup:llm` 설치 시 자유형, 없으면 룰 기반.)*
- **수동 자막 스타일** — 위치(3×3 앵커)·크기·색·외곽선·배경·폰트·키워드 강조색.
- **오버레이** — 스티커·GIF·이미지 패널, 미리보기에서 드래그/리사이즈, 키프레임 애니.
- **내 사전** — 자주 틀리는 고유명사 등록 → 전사 후 자동 교정.
- **챕터** — 무음·문장 경계로 `M:SS 제목` 타임스탬프 추출(유튜브용).
- **음성·TTS** — 텍스트 → 보이스오버 합성 + 믹스. 보이스 목록은 이 Mac에 설치된 음성(언어 표시)에서
  고르며 **한글을 입력하면 한국어 보이스(유나 등)로 자동 전환**됩니다. 만든 보이스 클립은
  타임라인 VOICE 트랙에서 **오버레이처럼 드래그로 위치·길이 조절**(선택 후 Delete로 삭제).
  한국어 보이스가 없으면 시스템 설정 ▸ 손쉬운 사용 ▸ 음성 콘텐츠에서 한국어(유나)를 받으세요.

**고급 모드로 실행:**
```bash
DAWN_ADVANCED=1 pnpm --filter @dawn-cut/desktop start
```

## 5. 단축키
`Space` 재생 · `←/→` ±0.1s · `Home/End` 처음/끝 · `Delete`/`Backspace` 선택 어절 컷 ·
`⌘Z`/`⇧⌘Z` 되돌리기/다시 · `⌘S` 저장 · `⌘O` 열기 · `⌘E` 내보내기 · `?` 도움말 · 어절 `⌘+클릭` 위치 점프

## 6. 내보내기 포맷
- **MP4**(H.264) — 자막 번인되어 있으면 포함, 아니면 끌 수 있는 소프트 자막 트랙(mov_text) 자동 첨부.
- **GIF** — 빠른 공유용.
- **SRT** — 자막 파일만(유튜브 업로드용).
- (mov/webm/HEVC·오디오-only·해상도 프리셋은 로드맵.)

## 7. 알아둘 것
- **오디오 없는 영상**은 자막 계열 불가(색보정·오버레이·컷·내보내기는 됨).
- **미리보기가 검게** 나오면 자동으로 작은 H.264 프록시로 변환해 재생한다(편집·내보내기는 원본 사용). GPU 가속을 꺼 호환성을 높였다.
- 모든 처리는 **로컬**. 자동 자막은 `pnpm setup:binaries`로 받은 whisper 모델 사용.
