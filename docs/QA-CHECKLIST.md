# dawn-cut QA 체크리스트

> 자동 QA 루프가 커버하는 것 + **사람이 직접 확인해야 하는 것**을 분리해 정리. (자연어 편집 제외)

## 0. 자동 QA 루프 재실행

```bash
pnpm --filter @dawn-cut/desktop build
npx playwright test --config playwright.qa.config.ts   # 8개 플로우 → output/qa/<flow>/
pnpm verify        # 단위+통합 28파일/39테스트
npx playwright test   # e2e 13개
```

산출물: `output/qa/<flow>/`에 스크린샷 + `result.json`(콘솔에러·상태 스냅샷). 현재 **8/8 통과, 콘솔에러 0**.

## 1. 자동으로 검증된 것 (✓ 기계 확인 완료)

- **가져오기/프록시/무음**: h264 네이티브 재생, AV1 → 프록시 변환 후 실제 영상 렌더(검은화면 X), 무음 영상은 자막 비활성+안내, 치우기 후 상태 초기화.
- **받아쓰기**: 한국어 47어절 전사, 텍스트컷으로 길이 감소(42.84→41.94s), 무음 제거로 클립 분할.
- **자막**: 앵커 9그리드·크기 슬라이더·세부스타일·프리셋·색 변경이 미리보기에 반영, 스타일팩 1클릭, 번인(overlays 0→47)+초기화.
- **효과**: 색보정 프리셋 즉시 프리뷰 반영.
- **오버레이**: 스티커/이미지 추가(2→4), 겹침 2행 표시, 드래그 이동, Delete 삭제(4→3).
- **TTS**: 한국어 보이스 자동 상위 노출, 생성→타임라인 보이스 블록, 드래그, 삭제.
- **내보내기**: MP4(23.6s, 이미지 오버레이 합성 확인)·GIF 유효 파일 생성.
- **드롭다운 한글**: 커스텀 KSelect로 모든 드롭다운 한글 정상 렌더(자모 깨짐 해결).
- **상태바/문구 한국어화**, 무음 0곳 시 버튼 비활성, Save/SRT 무자막 시 비활성+툴팁.

## 2. 🔴 직접 확인해야 하는 것 (자동화 불가 — 소리/움직임/체감)

자동 QA는 **정지 스크린샷 + 상태값**만 본다. 아래는 사람이 실제로 보고 들어야 한다:

### 재생 · 소리
- [ ] **TTS 한국어 음성이 자연스럽게 들리는가** (유나/에디 등). 발음·끊김·속도.
- [ ] **보이스오버 믹스**: 내보낸 MP4에서 음성이 *원하는 시점*에, 적절한 볼륨으로 섞이는가(타임라인에서 보이스 블록을 옮긴 위치대로).
- [ ] **재생 부드러움 / A·V 싱크**: ▶ 재생 시 끊김 없는지, 오디오-영상 싱크 맞는지.
- [ ] **스크럽/타임라인 클릭** 반응 속도 체감.

### 내보내기 결과물 (열어서 끝까지 보기)
- [ ] `output/qa/f8-export-project/export.mp4` 를 열어 **오버레이 위치·자막 가독성·색감**이 의도대로인지.
- [ ] **GIF** 품질/용량이 공유에 적당한지.
- [ ] 자막 번인 영상에서 **한글 자막이 깨지지 않고** 줄바꿈/위치가 맞는지.

### 조작 체감
- [ ] 오버레이·보이스 블록 **드래그/양끝 리사이즈**가 손맛 있게 따라오는지(자동 테스트는 좌표로만 끎).
- [ ] 같은 스티커 2개를 번호·색으로 실제로 구분하기 쉬운지.

### 내 실제 영상으로
- [ ] **강의 영상(0530/0601 폴더)** 으로 받아쓰기 정확도, 말버릇 감지, 무음 제거가 쓸만한지. (자동 QA는 짧은 ko-talk.mp4만 사용)
- [ ] 세로 영상/화면녹화(당근·오늘의집 등)에서 9:16 내보내기 크롭 결과.

> 확인 중 이상하면 그 장면을 캡쳐해서 알려주세요 — 같은 QA 루프로 재현·수정합니다.

### 자동 하이라이트 적극성 ✅ (해결)
- 예전엔 '말(speech) 길이 합'이 목표에 도달할 때까지 KEEP → 문장 사이 무음이 합산에서 빠져 말이
  적은 영상은 전부 KEEP→컷 0이었다. 이제 `timeline.durationProgram`(무음 포함) 기준으로 점수 낮은
  문장부터 **드롭**해 결과가 목표에 가깝게 줄어든다. 실측: ko-talk 42.84s + 목표 30초 → **25.78s로 컷**.
  단위 테스트(`highlight.test.ts`)에 '원본>목표면 실제 컷' 케이스 추가.

## 4. 빠른 자가 점검 순서 (5분)

1. 고급 모드로 실행: `DAWN_ADVANCED=1 pnpm --filter @dawn-cut/desktop start`
2. 강의 mp4 가져오기 → 재생 → ▶ 부드러운지
3. `자막 생성` → 어절 뜨는지 → 텍스트컷 → 무음 제거
4. `자막 입히기` → 미리보기 자막 → `내보내기 ▾ → MP4` → 결과 열어보기
5. `음성·TTS` → 한국어 문장 → `보이스 생성` → 소리 확인 → 타임라인 블록 드래그
6. `스티커·GIF` → 2개 추가 → 겹치게 → 드래그/Delete
