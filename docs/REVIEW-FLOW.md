# 검수 플로우 — 핵심 기능 중심

> dawn-cut의 **프로덕션 핵심 기능을 흐름 순서대로** 따라가며 직접 검수하는 문서.
> "기계가 이미 본 것"과 "사람이 꼭 봐야 하는 것"을 구분한다. (자연어 편집은 실험 기능이라 제외)
> 자동 QA 하네스(`tests/qa/`)가 매 기능을 실제 앱에서 구동·캡쳐하므로, 사람은 **소리·움직임·결과물**에 집중하면 된다.

## 실행
```bash
DAWN_ADVANCED=1 pnpm --filter @dawn-cut/desktop start   # 고급(전체) 모드
```
강의 영상(말소리 有)이 풀파워 대상. 화면녹화(무음)는 색보정·오버레이·9:16·내보내기까지 가능(자막 계열만 불가).

---

## 1. 핵심 플로우 (이 순서로 따라가며 검수)

각 단계: **할 일** → 👁 눈으로 / 🔊 소리로 / ✋ 손으로 확인할 점.

### ① 가져오기 · 미리보기
- 상단 **가져오기**(또는 미디어 패널에 드래그). 큰/AV1/고레벨 영상은 "미리보기 변환 중…"(스피너) 잠깐 뜬 뒤 재생.
- 👁 영상 첫 프레임이 뜨는가(검은 화면 아님). 미디어 카드 배지 `원본` / `프록시(미리보기용)` / `변환 중` 정확한가.
- 🔊✋ **▶ 재생** — 끊김 없는지, A·V 싱크 맞는지, 스크럽·타임라인 클릭 반응.
- 무음 영상이면 우측에 "오디오가 없어 자막 불가(색보정·오버레이·내보내기는 됨)" 안내가 뜨는가.

### ② 자막 생성 (받아쓰기)
- 우측 대본 패널 **🎙 자막 생성** → 한국어 어절이 대본에 나타남.
- 👁 받아쓰기 **정확도**(특히 내 실제 강의 영상). 좌측 카드 `자막 어절 N개` 표시.

### ③ 텍스트 편집 · 다듬기
- 대본 어절 클릭 선택 → `Delete` → 그 구간 컷(타임라인·길이 감소).
- **🧹 말버릇 제거** / **🔎 검수**(의심 어절 빨강 → 더블클릭 교정 → ↪ 다음) / 툴바 **✂ 무음 제거**(⚙로 민감도).
- 👁 무음 0곳이면 "제거할 무음 없음" + 버튼 비활성인가. 검수 하이라이트가 직관적인가.

### ④ 자막 스타일 (CapCut식 갤러리)
- 자막 카드: **스타일 갤러리**(12종 썸네일 — 기본/쇼츠/유튜브/예능/자막바/네온/미니멀/시네마/형광펜/팟캐스트/틱톡/한국어)에서 클릭.
- **애니메이션**: 없음 / 팝(커지며 등장) / 한 어절씩 / 타자기(글자씩) / 가라오케.
- **위치**(9-앵커) · **크기** · 세부 스타일(색·외곽선·배경·폰트·강조).
- **자막 입히기**(번인) → 미리보기/내보내기에 자막 합성. **초기화**로 되돌림.
- 👁 갤러리 썸네일이 실제 룩과 같은가(WYSIWYG). 미리보기 자막 == 번인 자막(글자·줄바꿈 일치)인가. 한글 안 깨지는가.

### ⑤ 효과 (색보정 · 비율)
- **✨ 자동 보정(1탭)** → "✓ 자동 보정 적용됨"(중립이면 "이미 노출이 좋아 변화가 작아요").
- **색보정 프리셋**(화사/따뜻/차갑/시네마…) — 프리뷰 즉시 반영.
- **비율 9:16 / 1:1** — 프리뷰에 **잘릴 영역 마스킹 + 세이프프레임** 가이드.
- 👁 색 변화가 프리뷰에 보이는가. 9:16 가이드가 실제 크롭 영역과 맞는가.

### ⑥ 오버레이 (스티커 · GIF · 이미지)
- **이모지 스티커** / **모션 스티커(움직이는 GIF 12종)** / **내 GIF·이미지 드롭존**(커스텀 업로드) / 텍스트 배지.
- 같은 스티커 2개는 **번호·색**으로 구분, 겹치면 타임라인에서 **2행**으로.
- ✋ 타임라인 블록 **드래그 이동 / 양끝 리사이즈**, 선택 후 `Delete` 삭제 — 손맛 있게 따라오는가.
- 👁 모션 GIF가 미리보기에서 실제로 움직이는가.

### ⑦ AI 보이스 (TTS)
- **보이스**(설치된 음성, 한국어 상단) · **스타일**(차분/보통/활기참) · **속도·톤** 슬라이더 · **▶ 미리듣기**.
- **보이스 생성** → 타임라인 VOICE 트랙에 블록. 드래그로 위치·길이 조절, `Delete` 삭제.
- 🔊 **미리듣기/생성 음성이 자연스러운가**(발음·속도·톤). 한글 → 한국어 보이스 자동 전환되는가.
- 엔진 배지: `macOS say`(기본) / `뉴럴(Piper)`(설치 시). → [한국어 뉴럴은 남은 트랙](REMAINING-TRACKS.md#a-한국어-뉴럴-tts-감정형-음색) 참고.

### ⑧ 자동 하이라이트 (롱폼 → 쇼츠)
- **✂️ 자동 하이라이트(60초)** → 핵심만 남겨 컷. 결과 알림: "핵심만 남겨 A→B" / "이미 짧아 컷 없음" / "대부분 핵심이라 더 못 줄임".
- 👁 길이가 실제로 줄고(예: 42.8s→25.8s), 남은 구간이 핵심인가.

### ⑨ 챕터 · 내 사전
- **📑 챕터** 추출(M:SS 제목) + 복사. **📒 내 사전**에 고유명사 등록 → 전사 자동 교정.

### ⑩ 내보내기 · 저장
- **내보내기 ▾**: **MP4**(자막 트랙 포함) / **GIF** / **SRT**(자막 있을 때만).
- **저장**(.dawn, 자막 없어도 가능) → **열기**로 복원. **되돌리기/다시하기**.
- 👁🔊 **내보낸 MP4를 끝까지 열어보기** — 자막 가독성·오버레이 위치·색감·**보이스 믹스 타이밍/볼륨**이 의도대로인가. GIF 품질/용량.

## 2. 기계가 이미 검증한 것 (자동 — 다시 안 봐도 됨)
`pnpm verify`(44) + `npx playwright test`(e2e 13) + `npx playwright test --config playwright.qa.config.ts`(QA 8 플로우, 콘솔에러 0):
- 가져오기/프록시(AV1·고레벨)/무음 안내, 네이티브·프록시 렌더(검은화면 아님), 치우기 초기화
- 받아쓰기·텍스트컷·무음 제거(클립 분할)
- 자막 프리셋/갤러리/애니(pop·typewriter 단위테스트)/위치/번인·초기화, 미리보기↔번인 일치
- 자동 보정(자막 없이도 적용)·색보정·9:16 마스크
- 오버레이 추가/겹침 2행/드래그/삭제, 모션 GIF 12종 로드·합성
- TTS 보이스 자동전환·속도(느림>빠름 1.5×)·생성·드래그·삭제, 엔진 배지
- 자동 하이라이트 실제 트림, 내보내기 MP4(오버레이 합성)·GIF 유효, 무자막 저장/열기
- 드롭다운 한글 정상(커스텀 KSelect), 상태바 한국어

## 3. 🔴 사람이 꼭 봐야 하는 것 (자동화 불가)
자동 QA는 **정지 스크린샷 + 상태값**만 본다. 아래는 직접:
- [ ] 🔊 TTS 음성이 자연스러운가(발음·끊김·속도·톤)
- [ ] 🔊 내보낸 MP4의 **보이스오버 믹스**가 원하는 시점·볼륨인가
- [ ] ▶ 재생 부드러움 / A·V 싱크 / 스크럽 반응
- [ ] 👁 내보낸 **MP4·GIF를 끝까지** 보기(자막 가독성·오버레이·색·비율)
- [ ] ✋ 오버레이·보이스 블록 드래그/리사이즈 손맛
- [ ] 📹 **내 실제 강의 영상(0530/0601)** 으로 받아쓰기 정확도·말버릇·무음 제거 체감
- [ ] 📹 세로/화면녹화(당근·오늘의집)에서 9:16 내보내기 크롭 결과

> 이상한 장면은 캡쳐해서 알려주면 같은 QA 루프로 재현·수정합니다.

## 4. 빠른 자가 점검 (5분 골든패스)
①가져오기→▶재생 → ②자막 생성 → ③무음 제거 → ④스타일 갤러리+애니+번인 → ⑦TTS 한국어 생성·미리듣기 → ⑥모션 스티커 추가·드래그 → ⑩MP4 내보내기·결과 확인.
